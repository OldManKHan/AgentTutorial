# OpenCode Agent 教程

## 项目简介

这是一个基于 [OpenCode](https://github.com/opencode/opencode) 开源项目的 AI Agent 完整教程。教程面向 Agent 小白，通过系统讲解理论与实践，帮助你从零理解 AI Agent 的工作原理，并学会如何开发自己的 Agent。

OpenCode 是一个开源的 AI 编程助手项目，具有 100% 开源、生产级质量、架构清晰、功能完整等特点，非常适合作为学习 Agent 开发的入门项目。

## 教程结构

本教程共分为 6 章，建议学习时间为 2-3 周：

1. **[第一章：Agent 是什么](./00-introduction.md)** - 介绍 Agent 的基本概念和 OpenCode 项目
2. **[第二章：工具系统 (Tools)](./01-what-is-agent.md)** - 深入理解 Agent 的核心组成
3. **[第三章：Agent Loop 核心循环](./02-tools.md)** - 学习 Agent 的执行流程
4. **[第四章：Prompt Engineering](./03-agent-loop.md)** - 掌握提示词工程技巧
5. **[第五章：高级主题](./04-prompt-engineering.md)** - 探索 Agent 开发的高级话题
6. **[第六章：实战项目](./05-advanced-topics.md)** - 通过项目实践巩固所学

## 学习前提

- 基础的 TypeScript/JavaScript 知识
- 了解 async/await 异步编程
- 有基本的 LLM API 调用经验（如 OpenAI API）

## 如何开始学习

1. 克隆本仓库到本地：
   ```bash
   git clone https://github.com/OldManKHan/AgentTutorial.git
   cd AgentTutorial
   ```

2. 按照章节顺序阅读 Markdown 文件，从 [00-introduction.md](./00-introduction.md) 开始。

3. 每个章节都包含理论讲解和代码示例，建议边读边实践。

4. 遇到问题时，可以参考 [OpenCode 项目](https://github.com/opencode/opencode) 的源代码。
